# Solana-etl for nft
Solana-etl collects data of nft collections, nfts and nft-trasactions through jsonrpc and save in mongodb.

## Requirements
1. Node.js >= 12
2. Mongodb

## Getting started
1. Install packages:   
`npm install`
3. Run migration:   
`npm run migrate`
1. Run project:   
`npm start`
